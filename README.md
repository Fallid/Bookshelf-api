# Bookshelf-Api

To install dependencies:

```bash
bun install
```
or
```bash
npm install
```

To run:

```bash
bun run start
```
or
```bash
npm run start
```

To Test API
- Using Postman
    - Import all files to use the test 
- Using Newman
    - ```bash 
        npm run test 
        ```
        or
        ```bash
        bun run test
        ```
